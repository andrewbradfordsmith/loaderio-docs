loader.io

loader.io is a simple and powerful cloud-based load testing tool. Using loader.io, you can quickly, easily, and freely simulate large numbers of connections to your web apps and ensure performance.

Getting Started

All you need to start testing is a valid email address and a server to test. Go to loader.io and create an account. Verify your email address and you'll be all set to create a test!
